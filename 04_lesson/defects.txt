ID: BUG-001
Название: Метод contains возвращает True для пустого символа
Описание: При вызове contains("SkyPro", "") метод возвращает True, хотя должен возвращать False.
Шаги воспроизведения:
1. Вызвать string_utils.contains("SkyPro", "")
Ожидаемый результат: False
Фактический результат: True
Серьёзность: Средняя
Приоритет: Средний
Окружение: Python 3.14.4, pytest 9.1.1
